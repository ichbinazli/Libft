Libft

libft, standart C kütüphanesi fonksiyonlarının yeniden implementasyonlarını ve C projelerinde kullanışlı ek fonksiyonları içeren özelleştirilmiş bir C kütüphanesidir. Hafif, yeniden kullanılabilir ve Unix tabanlı sistemlerle uyumludur.

Özellikler

Standart C kütüphanesi fonksiyonlarının yeniden yazımı (strlen, strcpy, atoi, vb.)

Ek yardımcı fonksiyonlar (ft_split, ft_strjoin, vb.)

Bellek ve string işleme araçları

Karakter tip kontrol ve dönüşüm fonksiyonları

Fonksiyonlar
String Fonksiyonları

ft_strlen – String uzunluğunu döndürür

ft_strcpy – String kopyalar

ft_strdup – String’in kopyasını heap üzerinde oluşturur

ft_strjoin – İki string’i birleştirir

ft_split – String’i verilen ayırıcıya göre böler

Bellek Fonksiyonları

ft_memset – Belleği belirli bir değer ile doldurur

ft_memcpy – Bellek bloklarını kopyalar

ft_memmove – Üst üste binen bellek bloklarını güvenle kopyalar

Karakter ve Dönüşüm Fonksiyonları

ft_atoi – String’i integer’a çevirir

ft_isalpha, ft_isdigit, ft_isalnum – Karakter tiplerini kontrol eder

ft_toupper, ft_tolower – Karakterleri büyük/küçük harfe çevirir

Fonksiyonların tam listesi ve kullanım örnekleri libft.h dosyasında mevcuttur.
